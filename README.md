<h4># Suvendu_MUVI_assignment
Automation tests using Rest Assured and TestNG<h4>
<br>
<h1>2. Description</h1>
This repository contains a short REST API automation assignment using Java, Rest Assured and TestNG.  
The goal is to demonstrate clean, reusable, framework‑ready code for GET and POST API automation.
<h1>3. Tech Stack</h1>
<ol>
<li>- Java 17</li>
<li>- Rest Assured</li>
<li>- TestNG</li>
<li>- Maven</li>
<li>- GitHub</li>
</ol>
<h1>4. Project Structure</h1>
src/
 └── test/java/com/muvi/api
      ├── base/        # Base setup (BaseTest)
      ├── pojo/        # POJO classes (Product.java)
      ├── tests/       # Test classes (GetApiTest, PostApiTest)
      └── utils/       # Payload builders, constants
<h1>5. Setup Instructions</h1>
1. Clone the repository:
   git clone https://github.com/suvendushekharsamal/Suvendu_MUVI_assignment.git

2. Navigate to the project folder:
   cd Suvendu_MUVI_assignment

3. Run tests using Maven:
   mvn test
<h1>6. Execution</h1>
- Tests are written in TestNG.
- Reports are generated in `test-output/` after execution.
- Example:
  mvn clean test
